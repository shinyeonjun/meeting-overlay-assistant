# 디렉토리 맵

이 문서는 현재 코드베이스에서 어디를 먼저 보면 되는지 빠르게 안내한다.

## 시작 순서

1. `server/app/main.py`
2. `server/app/entrypoints/`
3. `server/app/api/http/routes/`
4. `server/app/services/`
5. `server/app/infrastructure/persistence/postgresql/`
6. `client/overlay/src/main.js`
7. `client/web/src/main.jsx`
8. `client/shared/src/`
