# 스프린트 1 작업계획

상태: 기록용

이 문서는 초기에 기능 뼈대를 잡을 때의 작업 범위를 남긴 것이다.  
현재는 많은 항목이 완료됐고, 일부는 폐기되거나 방향이 바뀌었다.

## 당시 목표

- 세션 생성/종료
- 실시간 자막 흐름
- 이벤트 추출 뼈대
- 리포트 생성 경로

## 현재 기준으로 완료된 항목

- 세션 lifecycle 기본 구현
- 실시간 자막 UI
- 이벤트 저장/수정
- 수동 리포트 생성
- 세션별 리포트 폴더 구조

## 현재 기준으로 방향이 바뀐 항목

- 자동 리포트 생성 -> 수동 생성
- 실시간 다중 인사이트 -> 질문만 노출
- 리포트 탭 분리 -> 세션 탭 통합

이 문서는 더 이상 실행 계획 문서가 아니라, 프로젝트 초반 방향을 기록해두는 용도로 유지한다.
