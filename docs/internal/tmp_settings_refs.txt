현재 로컬 개발 시 자주 확인하는 설정 메모

1. STT 모델 경로
- STT_MODEL_PATH=backend/models/stt/faster-whisper-large-v3-turbo-ct2

2. system_audio backend
- STT_BACKEND_SYSTEM_AUDIO=hybrid_local_streaming_sherpa

3. mic backend
- STT_BACKEND_MIC=faster_whisper_streaming

4. 주의
- STT 경로 확인은 `uvicorn --reload`보다 일반 실행에서 먼저 본다.
- Hugging Face snapshot 절대 경로는 기본 경로로 쓰지 않는다.
