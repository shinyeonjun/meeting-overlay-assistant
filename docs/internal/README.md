# internal 문서 안내

`internal/`은 현재 진행 중인 메모와 내부 운영 참고를 두는 곳입니다.

중요한 점:

- 여기 문서는 현재 코드와 1:1로 고정된 공식 기준 문서가 아닙니다.
- 구조 기준은 `docs/architecture/`
- 제품 기준은 `docs/product/`
- 오래된 기록은 `docs/archive/`로 이동합니다.

## 현재 유지하는 문서

- [env_split_guide.md](env_split_guide.md)

## archive로 보낸 문서

- [스프린트1_작업계획.md](../archive/internal/스프린트1_작업계획.md)
- [MVP_개발백로그.md](../archive/internal/MVP_개발백로그.md)

## 사용 원칙

1. 현재 상태를 짧게 메모할 때만 `internal/`에 둡니다.
2. 공식 구조나 제품 기준으로 승격된 내용은 `architecture/`, `product/`로 옮깁니다.
3. 더 이상 직접 참조하지 않는 기록은 삭제 대신 `archive/`로 보냅니다.
