# report_v1 HTML 템플릿

이 템플릿은 한국식 회의록 양식을 기준으로 한 CAPS PDF 렌더링용 HTML/CSS 템플릿이다.

## 원칙

- PDF 레이아웃은 템플릿에서 고정한다.
- LLM은 STT 전사를 분석해 정본 데이터의 내용 필드를 채우고, 최종 레이아웃은 템플릿이 고정한다.
- 정본 데이터는 `ReportDocumentV1` 구조로 두고, HTML/PDF/웹 미리보기는 이 구조를 렌더링한 결과로 본다.
- 생성된 HTML/PDF/Markdown은 artifact storage에 저장하고, 템플릿은 repo에서 버전 관리한다.
- 전사 원문과 발화자 기반 인사이트는 검수용 데이터로 보되, 공유용 회의록 본문에는 노출하지 않는다.

## 섹션 구성

- 회의개요: 일시, 장소, 작성자, 작성일, 참석자, 안건
- 회의내용
- 결정사항
- 향후일정
- 특이사항
- 작성자

## 다음 연결 지점

HTML/PDF/Markdown 렌더러는 모두 같은 `ReportDocumentV1` 정본 데이터를 사용한다.
