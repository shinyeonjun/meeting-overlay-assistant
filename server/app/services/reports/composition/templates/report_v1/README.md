# report_v1 HTML 템플릿

이 템플릿은 한국식 회의록 양식을 기준으로 한 CAPS PDF 렌더링용 HTML/CSS 템플릿이다.

## 원칙

- PDF 레이아웃은 템플릿에서 고정한다.
- LLM은 문서 구조를 만들지 않고 섹션 내용을 채우는 역할만 맡는다.
- 정본 데이터는 `ReportDocumentV1` 구조로 두고, HTML/PDF/웹 미리보기는 이 구조를 렌더링한 결과로 본다.
- 생성된 PDF만 artifact storage에 저장하고, 템플릿은 repo에서 버전 관리한다.

## 섹션 구성

- 회의 메타데이터
- 회의내용
- 의결사항
- 회의결과
- 질문 및 확인사항
- 리스크 및 이슈
- 참고 전사

## 다음 연결 지점

현재 단계에서는 HTML 문자열 렌더링까지만 제공한다. 실제 PDF 생성 파이프라인 연결 시에는 Windows 로컬에서 안정적인 HTML-to-PDF 렌더러를 별도로 검증한 뒤 붙인다.
