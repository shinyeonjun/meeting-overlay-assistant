# Legacy Snapshot

이 디렉터리는 CAPS의 과거 구현을 참조용으로 보관한 아카이브입니다.

현재 공식 개발/운영 경로는 아래입니다.

- `server/`
- `client/`
- `shared/`
- `deploy/`

이 디렉터리의 하위 트리는 다음 성격으로 취급합니다.

- `legacy/backend/`: 과거 백엔드 스냅샷
- `legacy/frontend/`: 과거 프론트엔드 스냅샷

주의 사항:

- 신규 기능 개발은 이 경로에서 하지 않습니다.
- 현재 빌드/실행 기준 경로가 아니므로, 내부 스크립트나 설정이 즉시 실행 가능하다는 보장은 없습니다.
- 문서, 비교, 마이그레이션 참고용으로만 사용합니다.
